🏗Architecture

IoT Sensor Stream → FastAPI Ingest → Sliding Window (30 steps) ↓ Inference LSTM Autoencoder → Reconstruction Error + Isolation Forest → Anomaly Score ↓ if anomaly detected n8n Webhook → Slack / Email Alert

✨Key Features

LSTM Autoencoder

Encoder-decoder LSTM trained on normal sequences — anomalies produce high reconstruction error.

Isolation Forest Ensemble

Combines reconstruction error with Isolation Forest for robust multi-signal anomaly scoring.

Real-time Detection

Sliding window inference over 5-channel IoT data with <200ms end-to-end latency.

n8n Alert Automation

Anomaly events trigger n8n workflows that notify Slack/email with sensor context and confidence.

Confidence Scoring

Each detection returns a normalized anomaly score (0–1) for downstream severity triage.

Unsupervised Training

Trained entirely on normal operating data — no labeled anomaly examples required.

class LSTMAutoencoder(nn.Module):
      def __init__(self):
          super().__init__()
          self.encoder = nn.LSTM(N_FEATURES, HIDDEN_DIM, batch_first=True)
          self.decoder = nn.LSTM(HIDDEN_DIM, N_FEATURES, batch_first=True)

      def forward(self, x):          # x: [batch, seq_len, features]
          _, (h, _)  = self.encoder(x)
          repeated   = h.squeeze(0).unsqueeze(1).repeat(1, x.size(1), 1)
          out, _     = self.decoder(repeated)
          return out  # reconstruction of input sequence
